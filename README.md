# 🚀 100 Days of Code - C

Welcome to my **100 Days of Code** challenge!

This repository contains my journey of learning the C programming language by building one project every day.

## About

The goal of this challenge is to improve my programming skills through consistent daily practice.

During this journey, I focus on:

- Learning C programming
- Improving problem-solving skills
- Practicing algorithms and logic
- Using Git and GitHub daily
- Building a public portfolio

## Repository Structure


Each folder contains one independent project developed during the challenge.

## Technologies

- C

## Why I'm Doing This

I believe that consistency is the best way to improve as a programmer.

Building one project every day helps me strengthen my understanding of programming fundamentals while creating a portfolio that reflects my progress.
